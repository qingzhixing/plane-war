# Changelog

## [0.1.1] - 2026-03-13

### 战斗与数值
- **主炮攻速**：实际出手 **75 发/秒** 封顶；**理论射速**可继续随连击/速射升高，超出部分按 **每 1% → +0.02** 折算进主炮伤害（HUD **主炮伤害 ×× [+××]**）。
- **连击攻速**：10 / 25 / 50 / 100 连分档；**100 连后每再多 100 连 +5% 攻速**（倍率 +0.05，线性）。
- **Boss**：主线与续战 Boss **血量加厚**，避免高强化下过快蒸发。

### 续战与流程
- **续战 8 波**，第 **8 波为 Boss**；续关 **+1 稳态护盾**；进入续战前 **连续 3 次三选一**。
- 调试：**跳 Boss** 适配续战；局内设置 **连击 +10/50/100/500 与清零**。

### 武器与符卡
- **炸弹**为副武器：`PlayerBomb`、爆炸 **AoE**、爆炸帧清区域内 **敌弹**、飞行中可 **撞毁敌弹**；与玩家碰撞忽略。
- **符卡**：`spell_*` 与炸弹区分；符卡弹贴图与朝向；**逐发消弹**；可选 **自动符卡** 升级。
- **弓箭**：消弹与高伤；**回旋镖**：回一枚射一枚、词条与逻辑调整。
- 移除 **boss_hunter** 词条。

### HUD / UI
- 左侧 **LeftStatsVBox**：护盾层数、主炮（伤害含溢出、射速仅超上限时显示封顶、齐射分行）、副武器、符卡。
- **Combo** 置于波次旁；DPS / 分数布局调整；关于页展示 **版本号**（与 `config/version` 一致）。

### 修复与平台
- HUD **实际射速**不再因 `0.02` 下限误显示为 50/s；**player.gd** 子弹伤害缩进等修复。
- **擦弹**粒子 Android 可见与层级；**符卡区**与拖动不冲突（短按发动）。
