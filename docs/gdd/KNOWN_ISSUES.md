# 已知问题记录（临时）

- **HP HUD（心形图标）**
  - 现象：右上角心形图标位置与预期有偏差，受伤后图标有时全部消失。
  - 状态：实现已有基础逻辑（`scripts/hud.gd`）和图标资源（`art/ui/icon_hp.png`），但布局与刷新逻辑需要进一步调整。

- **玩家移动偶发卡顿**
  - 现象：拖拽玩家移动时，偶尔会感到轻微“卡一下”的不连续感。
  - 可能原因：当前基于指针位置差值直接修改 `global_position`，再在 `_update_movement` 中一次性 clamp，尚未对边缘与高频输入做平滑或节流。
  - 状态：后续需要专门调优移动与 clamp 逻辑，以保证持续平滑。
