# 08 UI/UX（MVP）

## 菜单

- 主菜单：开始 / 设置 / 退出
- 设置：
  - BGM/SFX 音量滑条（0–100%，默认 80%/100%）
  - BGM/SFX 静音开关
  - 震动开关
  - 画面缩放/适配（仅预留入口，可后续实现）

### 设置界面形态（MVP）

- 形式：全屏半透明遮罩 + 居中面板
- 入口：
  - 主菜单中的“设置”按钮
  - 局内可从暂停状态进入（例如在暂停界面增加“设置”按钮）
- 操作规则：
  - 拖动 BGM/SFX 滑条时，实时调用全局 `AudioManager`，更新线性音量（不打断当前 BGM）。
  - 切换静音开关时，仅修改对应通道的静音状态，不改变滑条的数值。
  - 关闭设置界面后，自动返回此前的菜单或局内状态（不额外改变暂停/继续状态）。

## 局内 HUD

- HP 与护盾（右上角，如「HP: 5  盾: 2」，护盾为 0 时不显示）
- 波次（如「第 1 波」）
- 经验条
- 暂停按钮（左上，**大号触控按钮**，文字“暂停 / 继续”）
- 符卡/炸弹按钮（若启用）

## 升级界面

- 3 张卡片：图标 + 名称 + 一句话效果
- 点击选择后立即回到战斗

## 失败与继续

- HP=0 → 弹出死亡提示遮罩，提示 “You Dead!”
- **继续游玩**：每局免费 1 次，恢复 50% HP（向上取整）+ 1 秒无敌，回到当前波次
- **重新开始**：大号触控按钮，重新加载本局
