# 08 UI/UX（MVP）

## 菜单

- 主菜单：开始 / 设置 / 退出
- 设置：BGM/SFX 音量、震动开关、画面缩放/适配

## 局内 HUD

- HP（右上角数字显示，如「HP: 3」，在 `Main.tscn` 中预先放置 `HUD/Root/HpLabel`，脚本仅更新文本）
- 经验条
- 暂停按钮（左上，**大号触控按钮**，文字“暂停 / 继续”）
- 符卡/炸弹按钮（若启用）

## 升级界面

- 3 张卡片：图标 + 名称 + 一句话效果
- 点击选择后立即回到战斗

## 失败与继续

- HP=0 → 弹出死亡提示遮罩（`GameOver/Root/...` 结构在场景中预先搭好），提示 “You Dead!” 并提供 **大号触控** “重新开始” 按钮
