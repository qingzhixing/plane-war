# 05b Boss 后继续挑战（威胁等级）

## 流程

- **首次击破 Boss**：弹出 **二选一**：**本局结算** / **继续挑战**。
- **本局结算**：与原先一致，进入结算界面（最高分/连击/DPS 等），可重开或回主菜单。
- **继续挑战**：
  - **威胁等级 +1**（局内累计，本局有效）。
  - **敌方增强**（见下表），**Boss 再次登场时**同步增强。
  - **玩家补偿**：**立即再触发一次三选一升级**（与波次结束相同界面）；之后从 **第 1 波** 重新刷怪，再次推进至 **第 8 波后 Boss**（同一只 Boss，数值随威胁等级提高）。
  - **得分**：全局评分乘区 **+8%/层**（与现有 `score_up` 等叠乘逻辑一致，在 Main 内累加）。

## 威胁等级 → 敌方倍率（MVP）

| 威胁等级 | 敌机/Boss HP 乘算（在波次 HP 之后） | Boss 敌弹基础速度乘算 |
|---------|-----------------------------------|----------------------|
| 0       | 1.00                              | 1.00                 |
| 每 +1   | ×1.12（复合：第 n 层为 1.12ⁿ）      | ×1.04（封顶约 1.35）  |

- 小怪仍按原 **波次** 公式缩血，再乘 **威胁 HP 倍率**。
- 可读性优先：弹速增幅温和，主要压力来自 **血量与战线拉长**。

## 验收

- Boss 死亡后必出现二选一；选结算必进原结算页。
- 选继续后必弹出升级三选一，关闭后第 1 波正常刷怪，第 8 波后再次 Boss。
- 第 2 周目起同波次小怪与 Boss 明显更耐打；Boss 弹幕略快。
