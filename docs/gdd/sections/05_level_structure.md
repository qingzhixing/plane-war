# 05 关卡结构（短波次快节奏）

## 波次与显示

- 局内 HUD 显示**当前波次**（如「第 1 波」），波次随时间或条件推进。
- 刷怪难度可随波次提升（生成间隔缩短、炮台/精英比例提高等）。

## 推荐节奏（MVP）

- 每一波持续时间较短（建议约 **10–18 秒**），偏向“快节奏刷波”体验。
- 每一波只刷出**少量敌人**（例如 3–8 个），强调“读弹 + 击破”的节奏，而非长时间农怪。
- 随波次推进，整体难度与紧张度逐步上升。
- Boss 战仍控制在 45–75 秒区间（可 1–2 阶段），作为若干波次之后的阶段目标。

## 刷怪与可读性原则

- 同屏高威胁弹源数量受控（建议 1–2 个）。
- 子弹形状/速度/密度保持“新手可躲”。
- 预警优先（激光/冲撞等危险技能先给提示）。

## 敌人刷新与波次升级（新增约定）

- 随着波次提升：
  - 敌人血量与威胁逐波上升（例如 HP 按固定比例随波次递增）。
  - 刷新模式可从单一类型逐渐过渡到“基础小怪 + 炮台/精英”组合。
- **升级触发方式**：
  - 升级不再依赖经验条，而是**以波次为节奏单位**：每通过若干波次（例如每 1 波或每 2 波），弹出一次三选一升级界面。
  - 升级时机应尽量安排在波次切换或清屏后，避免在激烈战斗中打断玩家操作。
