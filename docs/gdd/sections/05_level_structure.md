# 05 关卡结构（2–5 分钟波次制）

## 波次与显示

- 局内 HUD 显示**当前波次**（如「第 1 波」），波次随时间或条件推进（如每 45 秒 +1）
- 刷怪难度可随波次提升（生成间隔缩短、炮台比例提高等）

## 推荐节奏（MVP）

- **波次 1（教学/热身）**：低威胁小怪，子弹稀疏
- **波次 2（压力提升）**：数量提升或子弹稍快（仍可读）
- **波次 3（精英波）**：1 个精英 + 少量小怪
- **Boss 战**：45–75 秒（可 1–2 阶段）

## 刷怪与可读性原则

- 同屏高威胁弹源数量受控（建议 1–2 个）
- 子弹形状/速度/密度保持“新手可躲”
- 预警优先（激光/冲撞等危险技能先给提示）
