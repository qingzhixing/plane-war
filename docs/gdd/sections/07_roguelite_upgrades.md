# 07 Roguelite：升级三选一与强化池（MVP）

## 规则（已确认）

- 击杀敌人可用于统计表现（得分/连击/DPS 等），不再直接驱动升级触发。
- **升级触发改为按波次进行**：以“完成一定数量的波次”为节奏，定期弹出升级界面。
- 每次升级仍为三选一，强化可重复出现并叠加（建议上限 3 级）。

## 强化分类（MVP 目标 12–18 个）

- **主武器变体**：双发/三连/散射（散射角度相对集中，便于输出）/轻追踪
- **副武器/僚机**：小无人机跟随射击
- **生存**：护盾/抵挡 1 次伤害（已移除“受击无敌延长”类升级，以保持受击惩罚的一致性）
- **符卡/炸弹**：冷却缩短/范围增加/清弹增强
- **资源（可选）**：经验获取增加（Demo 可弱化）
