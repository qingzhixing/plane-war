# 07 Roguelite：升级三选一与强化池（MVP）

## 规则（已确认）

- 击杀敌人可用于统计表现（得分/连击/DPS 等），不再直接驱动升级触发。
- **升级由波次结束直接触发**：每波清场后默认弹出升级界面，无经验条。
- 每次升级仍为三选一，强化可重复出现并叠加（建议上限 3 级）。

## 强化分类（MVP 目标 12–18 个）

- **主武器解锁词条（新增核心）**：
  - 通过升级词条解锁新武器形态；首次解锁后立即切换到该武器。
  - 已解锁武器不再重复出现在候选池中，避免“空抽”。
  - 当前首批 3 把：
    - **基础子弹（默认）**：均衡输出，兼容多弹与聚焦构筑。
    - **弓箭**：高弹速、轻穿透，偏向中远程直线压制。
    - **回旋镖**：往返路径覆盖，可打二段命中，偏向持续压制。
- **主武器数值词条**：射速、伤害、弹速、弹道聚焦、多弹
- **副武器/僚机**：小无人机跟随射击
- **生存**：护盾/抵挡 1 次伤害（已移除“受击无敌延长”类升级，以保持受击惩罚的一致性）
- **符卡/炸弹**：冷却缩短/范围增加/清弹增强
- **表现向词条**：连击增益、Boss 特攻（避免仅“分数型”弱词条）

## 词条质量约束（新增）

- 减少“体感弱、反馈慢”的词条（如纯经验/纯小额分数加成），优先战斗内立即可感知收益。
- 每次三选一中至少保证 1 张为“直接战斗收益词条”（武器解锁或火力/符卡强化），避免选择压力过低。

## 后续扩展（可选）

- 引入稀有度（普通/稀有）
- 引入“互斥选择/构筑路线”（例如散射路线 vs 追踪路线）
