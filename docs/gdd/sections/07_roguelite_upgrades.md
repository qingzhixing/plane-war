# 07 Roguelite：升级三选一与强化池（MVP）

## 规则（已确认）

- 击杀获得经验
- 经验条满 → 弹出升级界面 → 三选一
- 强化可重复出现并叠加（建议上限 3 级）

## 强化分类（MVP 目标 12–18 个）

- **主武器变体**：双发/三连/散射/轻追踪
- **副武器/僚机**：小无人机跟随射击
- **生存**：护盾/受击无敌延长/抵挡 1 次伤害
- **符卡/炸弹**：冷却缩短/范围增加/清弹增强
- **资源（可选）**：经验获取增加（Demo 可弱化）
