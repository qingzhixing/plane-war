# 03 核心循环与操作（含继续游玩）

## 局外循环（Demo）

1. 进入游戏
2. 开始游戏（MVP：单机体直接进入）
3. 通关/失败结算
4. 返回主菜单再次开局

> Demo 阶段：不做局外永久养成（天赋树/装备/长期数值膨胀）。

## 局内循环（主循环）

- 击败敌人 → 获得经验
- 经验升级 → 三选一强化
- 波次推进 → Boss → 结算

## 失败与继续游玩（已确认）

- **HP 归零触发失败界面**，提供“继续游玩”
- **每局免费 1 次继续**
- 继续后建议效果（MVP 默认，可调整）：
  - 恢复至 **50% HP**（向上取整）
  - 获得 **1 秒无敌**
  - 回到当前波次继续（不回退进度，减少挫败）

## 操作与输入（竖屏、单手）

- **屏幕方向**：竖屏
- **移动**：单指拖拽，飞机跟随手指（建议加轻微缓动/死区，避免抖动）
- **射击**：自动射击（固定射速）
- **技能（建议 MVP 就做）**：一个“符卡/炸弹”按钮（清弹/护盾/高伤害其一），用于休闲兜底
